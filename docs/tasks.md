# 短链接服务任务清单

## 里程碑
- M1：基础能力可用（短链生成 + 重定向）
- M2：性能与缓存优化（Redis + 统计）
- M3：工程化完善（配置/日志/部署）

## 任务列表

### M1 基础能力
- 设计数据库表：links / visits / users（可选）
- 实现短链生成策略（Base62 或 Hash + 冲突处理）
- API：创建短链（POST /links）
- API：重定向（GET /:code）
- 基础错误码与响应格式
- 单元测试：生成策略与碰撞处理

### M2 缓存与统计
- Redis 缓存：code -> 原始链接
- 缓存穿透与失效策略
- 访问统计：PV/UV 计数
- 数据落库与异步化（可选）
- 压测脚本与结果记录

### M3 工程化
- 配置分环境（dev/staging/prod）
- 结构化日志与请求链路ID
- Dockerfile + docker-compose
- 接口文档与使用说明

## 验收标准
- 重定向接口 P95 < 50ms（本地压测）
- Redis 命中率 >= 90%
- 核心功能稳定运行 1 小时无异常
